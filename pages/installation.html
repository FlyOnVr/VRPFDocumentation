Project and Packages Setup

Import Package

Import VR Physics Framework into your Unity project.

Dependencies

1. Unity 6000.0.53f1+

2. XR Interaction Toolkit 2.5.4

3. URP

4. Animation Rigging 1.3.1

Project Setup

Head over to Tools → VRPF → Setup and click the “Run Full Setup" button.
